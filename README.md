# Python URL Shortener

## Description
A simple Python-based URL shortener that converts long URLs into short codes and stores them for retrieval.

## Features
- Generate short URL codes
- Store URL mappings in a file
- Retrieve original URLs

## Technologies Used
- Python
- File handling

## How to Run
python url_shortener.py

## Use Case
Demonstrates backend logic and data storage using Python.
